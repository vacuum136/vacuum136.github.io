以非root用户执行 Docker相关命令

安装完成后，执行如 docker ps命令时，提示没有权限。这是因为默认情况下，只有root 或者 docker组内的成员有权限执行 docker 命令。
如果不想每次都sudo执行命令，就需要将你添加到docker用户组内。该用户组在docker安装过程中自动被创建。
